Read the Web
with Python

Introduction

Let's look at how we can read information from a website using Python and Beautiful Soup. I've chosen to read the Amazon website since it's a popular e-commerce website.

Environment

I'm using Flask to serve the web app, so I've included the server_web.py file that calls the web reader's methods.

Product Page

The Web_Reader.read_product() method reads the Amazon web page and creates a Beautiful Soup object that contains all the tags and text in the webpage. We can now search through the web page represented in the Beautiful Soup object to find the information we need.

Here is what the Product Details section of the example page looks like:

In this example, we want to get the Best Sellers Rank (BSR) of the book, which, in this case, is #1,717. This number fluctuates daily - and even hourly - on the Amazon site relative to the other products available on the store, so you will probably see a different number when you look at that page.

When we inspect the product page, we'll see this HTML below:

We can find() the div we need in the soup object with this line of code:

feature_div = soup2.find(id='detailBullets_feature_div')

Once we have the div we need, we can find all unordered list items (ul) in the div and then loop through them looking for the span that contains the best sellers rank that we need for the product.

Category Page

Our Category search reads through the URLs on the page, then calls the Web_Reader.read_product() method, which goes to each product's page to find its best seller rank. You can see the results below:
